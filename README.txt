# XerocMod <3

This project is a mod for Minecraft. This mod is under development and will be updated in the future.

At the moment, the mod contains new armor, 2 new effects, new ore and ingots, as well as a new dimension that will be finalized. Also has a mod tab and tranlations(ru,en).

Current minecraft version is 1.18.1 but in future there will definitely be a transfer to other later versions.
